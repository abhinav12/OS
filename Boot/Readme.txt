To generate the assembly code from asm file use :
./create_boot_sector.sh [Assembly_File_Name]
To load the boot sector binary use : 
./load_boot_sector.sh [Binary_file_name] [Display_mode] #https://wiki.gentoo.org/wiki/QEMU/Options for more details
To exit the curses output screen, press Esc+2 to go to the main screen and enter q
